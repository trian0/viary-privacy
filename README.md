## 🇧🇷 Português

**Última atualização:** 16 de junho de 2026

### 1. Introdução

A Trian Apps desenvolveu o Viary como um aplicativo gratuito. Este documento informa os usuários sobre as práticas adotadas em relação a dados e privacidade.

Ao utilizar o Viary, você concorda com os termos descritos nesta Política de Privacidade.

### 2. Coleta de Dados

O Viary **não coleta, armazena nem transmite dados pessoais para servidores externos**. Todas as informações inseridas no aplicativo — viagens, checkpoints, fotos e valores gastos — ficam armazenadas exclusivamente no dispositivo do usuário.

### 3. Dados Gerados Localmente

Durante o uso do app, as seguintes informações são processadas localmente no dispositivo:

- **Localização GPS:** utilizada para rastrear a rota da viagem e calcular a distância percorrida. Não é compartilhada com terceiros.
- **Fotos:** acessadas da galeria do dispositivo para associar a checkpoints. Permanecem no dispositivo.
- **Dados de viagem:** nomes de locais, valores gastos e duração. Armazenados localmente via banco de dados Room.
- **País / Moeda:** detectados via requisição ao serviço ip-api.com para exibir o símbolo de moeda correto. Nenhum dado pessoal é enviado nessa requisição.

### 4. Permissões Solicitadas

- **Localização (ACCESS_FINE_LOCATION / ACCESS_COARSE_LOCATION):** necessária para rastrear a rota durante a viagem.
- **Galeria (READ_MEDIA_IMAGES):** necessária para adicionar fotos aos checkpoints.

### 5. Serviços de Terceiros

O aplicativo utiliza o serviço **ip-api.com** para identificação do país e moeda local. Essa requisição envolve o endereço IP do dispositivo, processado conforme a política de privacidade do próprio serviço.

O Viary não utiliza ferramentas de analytics, publicidade ou rastreamento de comportamento.

### 6. Segurança dos Dados

Como todos os dados são armazenados localmente, a segurança das informações depende das configurações de segurança do próprio dispositivo (senha, biometria, criptografia de armazenamento).

### 7. Privacidade de Crianças

O Viary não é direcionado a crianças menores de 13 anos e não coleta intencionalmente dados desse público.

### 8. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada periodicamente. Alterações significativas serão comunicadas por meio de atualização na página do aplicativo na Play Store.

Dúvidas, sugestões ou solicitações relacionadas a esta política:

## 🇺🇸 English

**Last updated:** June 16, 2026

### 1. Introduction

Trian Apps built the Viary app as a free application. This document informs users about the practices adopted regarding data and privacy.

By using Viary, you agree to the terms described in this Privacy Policy.

### 2. Data Collection

Viary **does not collect, store, or transmit personal data to external servers**. All information entered in the app — trips, checkpoints, photos, and expenses — is stored exclusively on the user's device.

### 3. Locally Generated Data

During app usage, the following information is processed locally on the device:

- **GPS Location:** used to track the trip route and calculate distance traveled. Not shared with third parties.
- **Photos:** accessed from the device gallery to associate with checkpoints. They remain on the device.
- **Trip data:** location names, expenses, and duration. Stored locally via the Room database.
- **Country / Currency:** detected via a request to ip-api.com to display the correct currency symbol. No personal data is sent in this request.

### 4. Permissions Requested

- **Location (ACCESS_FINE_LOCATION / ACCESS_COARSE_LOCATION):** required to track the route during a trip.
- **Gallery (READ_MEDIA_IMAGES):** required to add photos to checkpoints.

### 5. Third-Party Services

The app uses the **ip-api.com** service to identify the user's country and local currency. This request involves the device's IP address, processed according to the service's own privacy policy.

Viary does not use analytics, advertising, or behavior-tracking tools.

### 6. Data Security

Since all data is stored locally on the user's device, the security of the information depends on the device's own security settings (password, biometrics, storage encryption).

### 7. Children's Privacy

Viary is not directed at children under 13 and does not intentionally collect data from this audience.

### 8. Changes to This Policy

This Privacy Policy may be updated periodically. Significant changes will be communicated through an update on the app's Play Store page.
